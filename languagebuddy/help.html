Help

To start an activity or subject, select the button on the home screen. This should automatically load up the activity which you selected.

Once the activity has loaded, the instructions should appear on your screen when you begin the activity. If the activity doesn't load, click the button below it to download.

If you are trying to use a printable sheet, first enter the menu, then you will need to select the activity which you would like to print.

Once you have clicked on the printable sheet, it should load a page that looks like the one to the left, click the button which has been highlighted and follow the instructions seen on the print menu.
